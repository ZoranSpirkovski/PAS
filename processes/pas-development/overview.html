Orchestration Pattern

Hub-and-Spoke

Central orchestrator coordinates all agents. Multi-agent pipelines with clear phases — the orchestrator dispatches work, tracks status, and manages handoffs between phases. Sequential execution is enforced (phases run in order), with parallelism available within phases where agent I/O dependencies allow it.
